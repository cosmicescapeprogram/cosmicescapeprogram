## 第一部分：Linux文件和目录管理命令  
1.ls（列出目录内容）

ls 用于列出目录中的文件和文件夹。

ls ：列出当前目录下的所有文件

ls -l ：以详细信息（如权限、所有者、大小等）显示文件列表

ls -a ：显示所有文件，包括隐藏文件（以 . 开头的文件）

ls -lh ：文件大小以人类可读的格式（KB、MB）显示  
2. pwd（显示当前目录）

pwd 显示当前所在的目录路径。

3. cd（切换目录）
cd 用于在不同的目录之间移动。

cd 目录名 ：进入指定目录

cd .. ：返回上一级目录

cd ~ ：返回用户的主目录（/home/用户名）

cd - ：返回上一次所在的目录
